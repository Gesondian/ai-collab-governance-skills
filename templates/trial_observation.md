# trial_observation

Use this template to observe a governance rule before turning it into a hard requirement.

```md
trial_name:
date_label:
observer:

candidate_rule:
trigger_context:
sample_size:

observed_successes:
observed_failures:
agent_rationalizations:
evidence_gaps:
false_positives:
false_negatives:
observation_boundary:

recommended_change:
keep_lightweight:
promote_to_hard_gate:
do_not_promote_reason:

next_owner:
next_observation:
remaining_risks:
```

Allowed `recommended_change` values:

- `keep_as_wording_guidance`
- `revise_wording`
- `promote_to_template`
- `promote_to_hard_gate`
- `drop_rule`
